# [Noctalia Shell](https://github.com/noctalia-dev/noctalia-shell/) Plugins Repository

## Plugins

### "Day of Gaza Genocide"

Displays the number of days since the start of the So-Called "C̶e̶a̶s̶e̶F̶i̶r̶e̶" and the beginning of the the Genocide in 2023.
On click toggles between dates.
On hower shows extended variant in format: %Y years, %M months, %d days.

## Usage

1. Open Noctalia settings
2. Navigate to "Plugins" menu
3. Open "Sources"
4. Add `https://github.com/0opsies/noctalia-plugins`

Plugins from repository will be visible in the list.
